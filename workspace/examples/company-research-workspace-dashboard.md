# 公司研究 Workspace Dashboard

## Project

- Project ID: generated by Workspace.
- Research Question: 分析示例公司的收入质量和证据链。
- Runtime Workflow: company-research-workflow。

## Sessions

- Workspace Session 记录 runtime_id、workflow_id、intent 和 refs。

## Task Board

- Research Agent 选择方法论。
- Evidence Agent 登记证据包引用。
- Report Agent 生成报告草稿引用。

## Artifacts

- evidence_bundle: `schemas/evidence/evidence-chain.schema.json`
- research_report: `templates/report/research-report-template.md`

## Timeline

Timeline 记录 SESSION_OPENED、ARTIFACT_REGISTERED 和 SNAPSHOT_CREATED。

## Replay

Snapshot 可生成 Replay Plan，供 Verification Agent 复核。
