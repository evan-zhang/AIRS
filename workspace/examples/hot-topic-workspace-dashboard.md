# 热点主题 Workspace Dashboard

## Project

- Research Question: 分析示例热点主题的扩散路径和不确定性。
- Scope: topic。

## Sessions

- Runtime Workflow: hot-topic-workflow。

## Task Board

- 收集主题证据与反证。
- Review Agent 补充反方观点证据引用。

## Artifacts

- review_report: 记录复核意见和不确定性。

## Timeline

Timeline 帮助还原热点主题研究的证据采集、反方观点和复核节点。

## Audit

Collaboration Note 和 Artifact 登记都写入 Audit Log。

## Snapshot

热点主题研究常常存在快速变化的信息源，Workspace 可以在主题证据、反方观点和人工复核完成后创建 Snapshot。Snapshot 保留 runtime_plan_ref、artifact_refs 和 audit_log 引用，便于后续 Verification Agent 对同一研究问题进行 Replay。
