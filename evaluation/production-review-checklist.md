# Production Review Checklist（生产评审清单）

**归属 Milestone**：M6  
**免责声明**：本清单用于 AIRS 生产质量评审，不构成投资建议。

## 1. 必检项

- [ ] 是否包含 Scorecard。
- [ ] 是否引用 M2 Methodology。
- [ ] 是否引用 M3 Evidence Chain。
- [ ] 是否保留 M4 Prompt 输出或 Prompt 引用。
- [ ] 是否保留 M5 Skill trace 或 Skill 引用。
- [ ] 是否通过 Quality Gate。
- [ ] 是否包含反方观点、不确定性和风险披露。
- [ ] 是否包含“仅供研究参考，不构成投资建议”免责声明。

## 2. 强制拒绝

- [ ] 出现买入、卖出、持有、目标价或收益承诺。
- [ ] 核心结论没有 Evidence Card。
- [ ] Evidence Score 缺失。
- [ ] 权重总和无法解释。
- [ ] Evaluation Gate 为 FAIL。

## 3. 评审输出

评审输出必须包含：结论、失败项、条件通过项、修复建议、是否允许进入报告或 Benchmark。
