## 评估引擎概述

评估引擎负责对 AI Agent 投研能力进行评估，包括反方观点生成、不确定性标注、质量评估。

### 核心功能

- 反方观点生成
- 不确定性分析
- 质量评估
- Loop/Go 评估机制

---

## 评估类型

### 反方观点评估

评估报告是否包含实质性反方观点：
- 数据反方
- 逻辑反方
- 场景反方
- 时间反方

### 不确定性评估

评估不确定性标注是否合理：
- 数据不确定性
- 模型不确定性
- 假设条件

### 质量评估

评估研究报告质量：
- 证据完整性
- 逻辑一致性
- 结论有效性
