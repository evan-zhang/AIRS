# AIRS Master Skill

## Master Skill 简介

Master Skill 是 AIRS 系统的总控入口，负责：

- 接收用户研究意图
- 选择合适的研究方法论
- 协调各子 Skill 执行
- 输出结构化研究报告

---

## 激活条件

Master Skill 在以下情况下激活：

✅ **应该激活**：
- 用户提出投资研究相关问题
- 用户要求分析特定产业/公司
- 用户要求供应链分析
- 用户要求财报分析
- 用户要求事件影响分析

❌ **不应激活**：
- 用户要求荐股
- 用户要求自动交易
- 用户要求价格预测
- 用户要求投机策略
- 非投资研究相关问题

---

## 工作流程

### 标准研究流程

```
1. 意图识别
   - 解析用户研究问题
   - 识别研究类型（供应链/产业/个股/事件）
   - 确定研究范围
   
    ↓
2. 方法论选择
   - 根据研究类型选择方法论
   - 制定研究计划
   - 确定所需子 Skill
   
    ↓
3. 研究执行
   - 调用相关子 Skill
   - 执行具体研究任务
   - 收集研究发现
   
    ↓
4. 证据采集
   - 采集相关证据
   - 验证证据质量
   - 构建证据链
   
    ↓
5. 评分计算
   - 对研究对象进行多维度评分
   - 计算综合评分
   - 生成评分解释
   
    ↓
6. 反方观点生成
   - 生成反方观点
   - 分析结论不确定性
   - 标注风险点
   
    ↓
7. 报告生成
   - 按模板生成结构化报告
   - 包含免责声明
   - 输出最终报告
```

---

## 输入要求

### 必需输入

**研究问题**：
- 清晰、具体的研究问题
- 明确的研究对象（公司/产业/事件）
- 合理的研究范围

**示例**：
```
❌ 坏输入："分析一下这个股票"
✅ 好输入："分析 AI 服务器产业链中的供应链卡点"
```

### 可选输入

**研究约束**：
- 时间范围（如："近 3 年"）
- 地理范围（如："中国境内"）
- 数据源偏好（如："优先使用财报数据"）

**输出格式**：
- 报告详细程度（简报 / 详细报告）
- 报告语言（中文 / 英文）

---

## 输出要求

### 必需输出

**研究结论**：
- 核心观点清晰明确
- 基于证据，非主观臆断
- 标注不确定性等级

**证据链**：
- 完整的证据链
- 每个证据有来源
- 证据质量评分

**评分结果**：
- 多维度评分
- 综合评分
- 评分解释

**反方观点**：
- 实质性的反方观点
- 反方观点强度评估
- 反方观点证据支撑

**不确定性标注**：
- 结论不确定性等级
- 数据不确定性说明
- 假设条件说明

**免责声明**：
- 明确标注"不构成投资建议"
- 说明"仅供研究参考"
- 提示"投资有风险"

### 禁止输出

❌ **Master Skill 禁止输出**：
- 具体买入/卖出建议
- 目标价格预测
- "一定涨"/"一定跌"等绝对化表述
- 无证据支撑的主观判断
- 无免责声明的研究报告

---

## 证据要求

### 证据类型

Master Skill 要求使用以下类型证据：

1. **财报数据**：来自上市公司财报的原始数据
2. **行业报告**：来自券商、研究机构的行业分析
3. **新闻报道**：来自权威媒体的新闻
4. **公司公告**：上市公司官方公告
5. **政策文件**：政府发布的政策、法规
6. **专家观点**：行业专家的公开观点

### 证据质量

| 证据等级 | 描述 | 示例 |
|----------|------|------|
| A 级 | 官方、原始、可验证 | 上市公司财报原始数据 |
| B 级 | 权威来源、有分析 | 券商研究报告 |
| C 级 | 二手来源、需验证 | 媒体报道 |
| D 级 | 观点性质、仅供参考 | 专家观点 |

### 证据链要求

- 证据链完整，无断裂
- 证据之间有逻辑关联
- 关键结论有多个证据支撑
- 证据时效性符合研究要求

---

## 风险要求

### 风险识别

Master Skill 必须识别以下风险：

1. **数据风险**：数据来源可靠性、数据时效性
2. **模型风险**：分析方法局限性、假设条件
3. **市场风险**：市场波动、政策变化
4. **认知风险**：幸存者偏差、确认偏误
5. **技术风险**：技术迭代、竞争格局变化

### 风险披露

所有研究报告必须包含：

- 主要风险点
- 风险等级评估
- 风险影响分析
- 风险应对建议（如有）

---

## 反方观点要求

### 反方观点类型

Master Skill 必须生成以下类型反方观点：

1. **数据反方**：用不同数据来源得出的相反结论
2. **逻辑反方**：对推理过程的质疑
3. **场景反方**：在不同情景下结论可能不成立
4. **时间反方**：从不同时间维度看结论的局限性

### 反方观点强度

| 强度 | 描述 | 要求 |
|------|------|------|
| 强 | 有充分证据支撑的反方观点 | 必须有 B 级以上证据 |
| 中 | 有一定依据的反方观点 | 至少有 C 级以上证据 |
| 弱 | 推测性反方观点 | 明确标注为推测 |

---

## 报告格式

### 报告结构

Master Skill 生成的报告必须包含以下部分：

```markdown
# 研究报告标题

## 1. 核心观点
[清晰的核心观点，不超过 200 字]

## 2. 研究方法
[使用的方法论、数据来源、分析框架]

## 3. 证据链
### 3.1 关键证据
[关键证据列表]
### 3.2 证据链图谱
[证据之间的逻辑关系]

## 4. 评分结果
### 4.1 多维度评分
[各维度评分]
### 4.2 综合评分
[综合评分及解释]

## 5. 反方观点
### 5.1 主要反方观点
[反方观点列表]
### 5.2 反方观点强度评估
[强度评估]

## 6. 不确定性标注
### 6.1 数据不确定性
[数据不确定性说明]
### 6.2 模型不确定性
[分析方法局限性]
### 6.3 假设条件
[关键假设]

## 7. 风险提示
[主要风险点]

## 8. 免责声明
[标准免责声明]
```

### 子 Skill 列表

Master Skill 可以调用以下子 Skill：

| Skill | 用途 | 目录 |
|-------|------|------|
| hot-topic | 热点主题分析 | skills/hot-topic/ |
| industry | 产业分析 | skills/industry/ |
| supply-chain | 供应链分析 | skills/supply-chain/ |
| chokepoint | 卡点分析 | skills/chokepoint/ |
| financial | 财报分析 | skills/financial/ |

| committee | 投资委员会 | skills/committee/ |

### 调用规范

- 调用前检查 Skill 可用性
- 传递清晰的输入参数
- 处理 Skill 返回的错误
- 验证 Skill 输出质量

## 禁止事项

### 严格禁止

❌ **Master Skill 严格禁止**：
1. 直接荐股："建议买入 XXX"
2. 价格预测："目标价 100 元"
3. 绝对化表述："一定涨""必然涨"
4. 无证据结论："我觉得这个好"
5. 无免责声明报告
6. 无反方观点报告
7. 无不确定性标注报告

### 违规处理

如果检测到违规内容：

1. 拒绝生成报告
2. 返回错误提示
3. 说明违规原因
4. 给出修正建议

---

## 质量标准

### Master Skill 质量标准

- 研究结论有充分证据支撑
- 证据链完整，可追溯
- 包含实质性反方观点
- 不确定性标注合理
- 有完整免责声明
- 符合报告模板规范

### 评估维度

| 维度 | 标准 |
|------|------|
| 证据质量 | A/B 级证据占比 > 70% |
| 反方观点强度 | 至少有一个强反方观点 |
| 不确定性标注 | 所有关键结论都有不确定性标注 |
| 报告完整性 | 包含所有必需章节 |
