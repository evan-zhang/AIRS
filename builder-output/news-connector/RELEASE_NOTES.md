# News Connector Release Notes

## Summary

新增 News Connector Feature Package，用于把新闻事件转化为 AIRS 可审查的结构化事件与 Evidence 引用。该 Package 支持新闻来源追溯、事件归一化、反方来源、缺失证据和不确定性标注。
