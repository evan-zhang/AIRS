# Financial Statement Anomaly Analysis（财报异常分析）

**归属 Milestone**：M2 Methodology Core  
**适用层级**：Methodology Layer → Research Skill Layer → Evaluation Layer  
**免责声明**：本文档仅用于财务研究方法定义，不构成审计意见，不构成投资建议，也不提供交易指令。

## 1. Purpose（目的）

财报异常分析用于识别上市公司财务报表中可能反映经营变化、会计处理变化或风险暴露的异常信号。目标是生成可验证的研究线索，而不是直接判定公司存在违规或给出投资结论。

## 2. Theory（理论基础）

理论基础来自财务报表分析、应计质量、现金流匹配和异常检测。收入、利润、现金流、资产负债表和附注之间应存在基本一致性；若出现持续背离，可能指向经营质量变化、会计估计变化或一次性因素。

## 3. Background（背景）

AIRS 后续需要处理公司研究和风险分析。财报异常方法为 `financial`、`risk` 和 `evidence` Skill 提供统一框架，避免 Agent 只复述财务指标而不检查指标间关系。

## 4. Applicable Scenarios（适用场景）

- 用户要求分析公司财报质量、盈利质量或异常科目。
- 报告需要解释利润增长与现金流、应收、存货或负债变化之间的关系。
- 需要为风险章节提供财务证据。

## 5. Non-applicable Scenarios（不适用场景）

- 公司财务数据不可得或口径严重不一致。
- 用户要求作出法律定性、审计结论或违规指控。
- 用户要求根据财报异常直接买卖证券。

## 6. Inputs（输入要求）

- 公司、期间和报表口径。
- 利润表、资产负债表、现金流量表和附注摘要。
- 同业可比指标和历史序列。
- 重大会计政策、一次性项目和并购重组信息。

## 7. Outputs（输出规范）

- 异常信号清单：科目、变化幅度、可能解释和证据。
- 三表勾稽检查：利润、现金流、资产负债变化的关系。
- 反证和替代解释：季节性、会计政策、业务模式或行业周期。
- 风险等级、置信度和免责声明。

## 8. Workflow（工作流程）

1. 固定报表口径和比较期间。
2. 计算关键指标的同比、环比、结构占比和同业偏离。
3. 检查收入、利润、现金流、应收、存货和资本开支之间的勾稽关系。
4. 识别异常，并为每个异常生成经营解释、会计解释和风险解释。
5. 查找附注、公告和管理层说明作为证据或反证。
6. 输出异常等级、置信度和后续验证问题。

## 9. Required Evidence（必需证据）

- 原始财报或公司公告中的财务数据。
- 至少三期历史数据或可比公司数据。
- 附注、管理层讨论或审计意见中与异常相关的说明。
- 与行业周期或业务模式相关的背景证据。

## 10. Counter Evidence（反证要求）

- 异常是否由季节性、一次性项目、并表变化或会计政策调整解释。
- 现金流背离是否由业务扩张、客户结构或结算周期导致。
- 存货或应收增长是否与订单和收入增长匹配。
- 同业是否出现类似变化。

## 11. Failure Cases（失效场景）

- 用单期数据判断长期趋势。
- 将会计异常直接等同于财务造假。
- 忽略行业商业模式导致的正常科目差异。
- 未读取附注和审计意见。

## 12. Confidence（置信度评估方法）

置信度取决于数据完整性、历史一致性、同业对比、附注解释质量和反证强度。若异常可由公开披露合理解释，应降低风险结论强度；若多个科目持续背离且缺乏解释，置信度提高。

## 13. Benchmark Mapping（基准测试映射）

Benchmark 应设置真实异常、可解释异常和噪声异常三类案例。评测 Agent 是否能进行三表勾稽、引用原始财报、给出替代解释，并避免作出未经证实的违规判断。

## 14. Future Prompt Mapping（未来 Prompt 映射）

Prompt 应包含财务数据抽取、异常检测、附注验证、同业比较和风险表述模块。输出必须明确“研究线索”而非“审计结论”。

## 15. Future Skill Mapping（未来 Skill 映射）

映射到 `financial`、`evidence`、`risk`、`valuation` 和 `report` Skill。Financial Skill 识别异常，Evidence Skill 绑定来源，Risk Skill 评估影响。

## 16. Future Score Mapping（未来评分映射）

评分维度包括异常幅度、持续性、三表一致性、同业偏离、解释充分性、现金流质量和证据等级。评分用于风险研究，不构成投资评级。
