# Theme Expansion Analysis（主题扩散分析）

**归属 Milestone**：M2 Methodology Core  
**适用层级**：Methodology Layer → Research Skill Layer → Benchmark Layer  
**免责声明**：本文档仅用于研究方法定义，不构成投资建议，不提供交易指令或价格预测。

## 1. Purpose（目的）

主题扩散分析用于研究一个产业、政策、技术或事件主题如何从核心资产、核心环节向上下游、替代链、应用场景和跨行业方向扩散。该方法论帮助 Agent 区分真实扩散、叙事扩散和市场情绪扩散。

## 2. Theory（理论基础）

理论基础来自创新扩散理论、产业链传导和注意力经济。主题扩散通常经历核心催化、证据确认、受益链条延展、边际预期过度和回归基本面等阶段。有效扩散必须能在业务、订单、成本或政策中找到证据。

## 3. Background（背景）

AI、机器人、新能源和创新药等主题容易出现从核心技术向材料、设备、软件、服务和基础设施扩散的现象。AIRS 需要让 Agent 判断扩散路径是否有产业逻辑，而不是机械罗列概念相关公司。

## 4. Applicable Scenarios（适用场景）

- 用户询问某个热点主题会影响哪些行业、环节或公司类型。
- 需要建立“主题 → 产业链 → 受益机制 → 证据”的研究地图。
- 需要识别主题处于早期、中期、后期还是过热阶段。

## 5. Non-applicable Scenarios（不适用场景）

- 研究对象是单一公司治理问题，主题传导不是主要矛盾。
- 缺乏可验证催化或产业变化，只有社交媒体热度。
- 用户要求按主题直接推荐股票或给出买卖清单。

## 6. Inputs（输入要求）

- 主题定义：明确技术、政策、事件或需求变化的边界。
- 起点证据：说明主题为何成立，例如政策文件、产品发布、订单变化或资本开支。
- 扩散范围：行业、区域、时间窗口和上下游边界。
- 输出深度：主题地图、候选链条、风险或 Benchmark 用例。

## 7. Outputs（输出规范）

- 主题定义和核心假设。
- 扩散路径分层：核心层、直接受益层、间接受益层、伪相关层。
- 每条路径的证据、反证和置信度。
- 主题阶段判断与不确定性说明。

## 8. Workflow（工作流程）

1. 固定主题边界，避免把宽泛标签当作研究对象。
2. 找到主题成立的一级证据和核心驱动变量。
3. 绘制扩散路径，标注传导机制和可观察指标。
4. 对每条路径进行证据验证，剔除无业务联系的概念映射。
5. 生成反共识和风险检查，评估过度扩散可能。
6. 输出结构化主题地图和可跟踪指标。

## 9. Required Evidence（必需证据）

- 政策、技术、订单、资本开支、客户需求或行业数据中的至少两类证据。
- 产业链关系证据，证明扩散节点与主题存在业务关联。
- 时间序列证据，用于判断主题是否正在增强或衰减。
- 明确证据等级和来源。

## 10. Counter Evidence（反证要求）

- 是否存在仅名称相似、业务无关的伪主题标的。
- 主题受益是否已被竞争、价格下行或成本上升抵消。
- 催化因素是否一次性、短周期或不可重复。
- 核心假设是否依赖未验证的需求扩张。

## 11. Failure Cases（失效场景）

- 把新闻热度当作产业扩散证据。
- 忽略主题链条中的价值分配和利润率差异。
- 只输出概念列表，不说明传导机制。
- 将主题热度直接转化为投资建议。

## 12. Confidence（置信度评估方法）

置信度取决于催化证据强度、传导链条清晰度、业务相关性、数据时效性和反证压力。若主题只存在舆情热度而没有订单、政策或财务验证，应给低置信度。

## 13. Benchmark Mapping（基准测试映射）

Benchmark 应包含真实扩散、伪扩散和过度扩散案例。评测重点是 Agent 是否能区分核心层与边缘层，是否能要求证据，而不是简单生成热门概念清单。

## 14. Future Prompt Mapping（未来 Prompt 映射）

Prompt 应包括主题定义、扩散路径生成、证据验证、伪相关剔除和主题阶段判断。输出格式必须强制包含反证和免责声明。

## 15. Future Skill Mapping（未来 Skill 映射）

可映射到 `hot-topic`、`industry`、`supply-chain`、`evidence` 和 `report` Skill。主题 Skill 负责扩散地图，Evidence Skill 负责验证每个节点。

## 16. Future Score Mapping（未来评分映射）

评分维度建议包括主题强度、业务相关度、传导距离、证据质量、持续性、拥挤度和反证强度。评分仅用于研究排序，不构成投资评级。
