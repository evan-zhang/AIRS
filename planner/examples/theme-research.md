# Theme Research Plan Example

免责声明：本文档仅用于 AIRS 工程开发与研究质量控制，不构成投资建议，不提供荐股、自动交易、交易指令、目标价或收益承诺。

## Goal Analysis

目标是分析 AI 算力主题扩散路径，Planner 将其识别为 Theme Research。

## Scope

范围包含主题信号、扩散链路、反共识证据、风险因素和报告交付。范围排除热点追涨表述和未验证结论。

## Required Connectors/Methodologies/Skills/Runtime

Connectors：market_news、research_report、policy。Methodologies：theme-expansion、counter-consensus、evidence-chain。Skills：hot-topic、evidence、risk、report。Runtime 必须带 Planner Gate。

## Expected Evidence/KG

至少包含需求信号、供给约束、政策事件和反方证据四类 Evidence Card。KG 连接主题、行业、公司角色、证据和风险。

## Execution Order

goal_analysis -> scope -> methodology_selection -> connector_plan -> evidence_plan -> skill_plan -> knowledge_graph_plan -> score_plan -> report_plan -> review_plan -> runtime_plan。

## Estimated Cost/Time

预计 81 分钟，成本单位约 23，最大并发 3。

## Expected Deliverables

主题扩散计划、Evidence Chain 规划、KG 规划、Scorecard 规划、报告提纲和复核清单。

## Confidence

主题研究初始置信度中等，需通过反共识证据和缺失证据提升可复核性。

## Risks

舆情噪声、证据滞后、主题边界过宽、方法论适配偏差。
