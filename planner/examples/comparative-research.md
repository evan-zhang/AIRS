# Comparative Research Plan Example

免责声明：本文档仅用于 AIRS 工程开发与研究质量控制，不构成投资建议，不提供荐股、自动交易、交易指令、目标价或收益承诺。

## Goal Analysis

目标是比较云端 AI 与边缘 AI 的证据链和风险，Planner 将其识别为 Comparative Research。

## Scope

范围包含可比对象、指标口径、证据链、风险和反方观点。范围排除单一结论排序和交易动作。

## Required Connectors/Methodologies/Skills/Runtime

Connectors：company_filings、industry_report、market_news。Methodologies：valuation、financial-anomaly、management-quality。Skills：financial、valuation、evidence、report。Runtime 必须由 Planner 触发。

## Expected Evidence/KG

Evidence Plan 需要双方可比数据、反方证据、缺失证据和口径说明。KG 连接两个研究对象、指标、证据和风险。

## Execution Order

goal_analysis -> scope -> methodology_selection -> connector_plan -> evidence_plan -> skill_plan -> knowledge_graph_plan -> score_plan -> report_plan -> review_plan -> runtime_plan。

## Estimated Cost/Time

预计 81 分钟，成本单位约 23，最大并发 3。

## Expected Deliverables

对比研究计划、可比指标表、Evidence Plan、KG Plan、Scorecard Plan、报告提纲。

## Confidence

置信度取决于两个对象是否存在可比口径和同周期数据。

## Risks

指标口径不一致、数据周期错位、证据强度不平衡、比较结论被过度解读。
