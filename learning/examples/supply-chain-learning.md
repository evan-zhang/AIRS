# Supply Chain Learning Example

## Feedback Input

- 供应链研究识别了卡点，但缺少上游替代路径证据。
- Outcome 复核发现原报告对扩产难度的置信度偏高。

## Pattern Mining

Learning Engine 将反馈归并为 `investment_engine/missing_chokepoint_evidence`，并识别 `score/outcome_variance`。

## Rule Generation

生成规则候选：供应链卡点结论必须同时引用瓶颈证据、替代路径证据、扩产周期证据和反方证据。

## Optimization Suggestions

- Methodology：增加替代路径验证步骤。
- Score：提高缺失反方证据的惩罚。
- Skill：供应链 Skill 增加 chokepoint evidence checklist。
