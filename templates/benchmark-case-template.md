## 4. 验收标准

### 4.1 通过标准

**核心观点**：
- ✓ 核心观点清晰明确
- ✓ 关键结论符合预期

**证据链**：
- ✓ 证据类型符合要求
- ✓ 证据质量达标
- ✓ 证据链完整

**评分**（如适用）：
- ✓ 各维度评分在期望范围内
- ✓ 综合评分在期望范围内

**反方观点**：
- ✓ 包含必需的反方观点类型
- ✓ 反方观点有实质内容

**不确定性标注**：
- ✓ 包含必需的不确定性标注

**格式**：
- ✓ 符合报告模板格式
- ✓ 包含免责声明

### 4.2 评分细则

| 检查项 | 权重 | 通过标准 |
|--------|------|----------|
| 核心观点准确性 | 20% | 与期望一致 |
| 证据完整性 | 30% | 覆盖必需证据 |
| 证据质量 | 15% | 质量达标 |
| 评分准确性（如适用） | 15% | 在期望范围内 |
| 反方观点强度 | 10% | 有实质内容 |
| 格式规范性 | 10% | 符合模板 |

**总分 >= 80 分**：通过

**总分 60-79 分**：部分通过

**总分 < 60 分**：不通过
