# AIRS Templates 目录

本目录包含 AIRS 项目的模板文件。

---

## 目录结构

```
templates/
├── README.md                      # 本文件
├── report-template.md             # 研究报告模板
├── evidence-card-template.md      # 证据卡模板
├── score-card-template.md         # 评分卡模板
├── benchmark-case-template.md     # 基准测试用例模板
└── skill-template.md              # Skill 开发模板
```

---

## 模板分类

### 研究报告模板

标准的研究报告格式，包含：
- 核心观点
- 研究方法
- 证据链
- 评分结果
- 反方观点
- 不确定性标注
- 风险提示
- 免责声明

**归属 Milestone**：M5

---

### 证据卡模板

标准的证据卡格式，包含：
- 证据类型
- 证据来源
- 证据内容
- 证据时效
- 证据质量

**归属 Milestone**：M3

---

### 评分卡模板

标准的评分卡格式，包含：
- 评分维度
- 评分结果
- 评分解释
- 评分置信度

**归属 Milestone**：M4

---

### 基准测试用例模板

标准的测试用例格式，包含：
- 用例说明
- 输入格式
- 期望输出
- 元数据

**归属 Milestone**：M7

---

### Skill 开发模板

标准的 Skill 开发格式，包含：
- Skill 说明
- 激活条件
- 输入输出
- 工作流程
- 依赖关系

**归属 Milestone**：M2

## 模板使用

### 使用流程

1. 复制对应模板
2. 填充实际内容
3. 验证格式正确性
4. 提交使用

### 自定义扩展

- 可基于模板自定义
- 保持核心结构
- 遵循命名规范
